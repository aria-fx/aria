Asset Registry for Intelligent Agents

ARIA Framework

A practical reference architecture for classifying, governing, and operationalizing enterprise AI assets.

View Repository Read Docs

What ARIA Solves

  • Standardize AI asset metadata using OASF records and governance overlays.
  • Publish and discover assets through a GitHub and OCI-based marketplace model.
  • Enforce compliance with Purview-integrated policy controls and auditability.

Architecture Layers

Metamodel

OASF entities, skills, domains, and modules define a shared language for AI assets.

Marketplace

GitHub repositories + OCI artifacts make assets versionable, composable, and discoverable.

Governance

Purview and OASF overlays provide policy enforcement, lineage context, and controls.
